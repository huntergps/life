# Product Definition — Galapagos Wildlife

## What is it

A mobile application for exploring and documenting the wildlife of the Galapagos Islands.

## Target Audience

1. **Tourists** — Identify species, learn about wildlife, record sightings
2. **Naturalist guides** — Field reference tool, species catalog, trail maps
3. **Park rangers (PNG)** — Field data collection, site monitoring, trail management
4. **Researchers** — Species observation data, citizen science contributions

## Sub-products within the system

1. **Public App** — Species catalog, sighting recording, favorites, badges
2. **Field Tools** (Beta) — Interactive map, trail tracking, site editing, AR camera, Photo ID
3. **Admin/Editorial** — Species data management, image curation, user role management
4. **Watch Companion** — Quick sighting recording, species browsing on Apple Watch

## Core Principles

- Offline-first: everything works without internet
- Open-source AI: no proprietary cloud APIs for identification
- Bilingual: Spanish and English throughout
- Conservation-focused: IUCN status, threats, endemic species highlighted
